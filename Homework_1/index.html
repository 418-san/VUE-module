Полив бонсай

Пересадка — важная операция, способствующая поддержанию здоровья растения. При пересадке бонсай контейнер обычно не меняют. Только в исключительных случаях, например, при желании улучшить композицию, используют контейнер другой формы, размера или цвета.

Основная цель пересадки — заменить часть почвы свежей, удалить загущенные угнетенные корни. Для этого вынимают растение из контейнера и вырезают снизу часть корневой системы. Очищают почву с корней так, чтобы они выступали из остающегося кома на 1-2 см. После этого делают новый дренаж, подсыпают свежую почву и укрепляют растение в прежнем положении. После пересадки растение поливают.
